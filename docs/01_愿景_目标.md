# 愿景 / 目标（MVP）
## 愿景
做一个**轻量稳定的网页小游戏聚合站**：分类清晰、进入即玩、后续可逐步演进为专题/社区站。

## MVP 目标（本轮）
- 首页：推荐 + 最近上架（从 `data/games.json` 自动渲染）
- 分类页：RPG / 动作 / 休闲（从 `games.json` 过滤渲染）
- 详情页：`game.html?id=<id>`（支持 canvas 演示 & iframe 外链）
- 统一导航（自动插入，高亮当前页）
- 无后端，GitHub Pages 托管

## 非目标（后续考虑）
- 登录、评论、排行、成就系统
- 复杂搜索/推荐算法
- 自研后端
